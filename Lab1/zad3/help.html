Why should you learn Java?

Java is the most popular language:

# Language Popularity
1 Java 23%
2 Python 22%
3 C++ 12%

Versatility

The last point is versatility. In Java, you can write an application to your computer without any problems, and thanks to the fact that applications written in this technology are run on a virtual machine, once written, you can run on windows, macros and Linux without any problem. You may associate Minecraft - it is also written in Java and you can write additional extensions to it. However, the matter does not end there. Java is also the default language used on the android system. So if you want to create applications for the most popular mobile system in the world, Java will be a great choice. The last place where Java is used is server environments and large business applications. In fact, although you probably do not realize it, Java is the most used one. You can imagine that if you click the "send transfer" button on your bank's website or "buy now" on the allegro, it is very likely that the application written in Java is processing your request. Thanks to the fact that Java is a very efficient, safe and scaling technology, large companies are focusing on it as the main technology.
